# PactRide site assets

The GitHub Pages site is intentionally dependency-free:

- `../index.html` contains the accessible semantic structure and project content.
- `styles.css` contains the responsive visual system.
- No analytics, trackers, external fonts, cookies, JavaScript frameworks, or third-party CDNs are used.

Design goals:

- editorial clarity rather than startup hype;
- fast loading on low-power and mobile devices;
- strong accessibility and reduced-motion support;
- honest separation between protocol research and a production ride service;
- direct links into the specification, threat model, governance, and contribution queue.
